如何找回您提现BEP20 而误选ERC20 的Token？

背景​

在提币的过程中，因为ERC20 和 BEP20的地址规则是一样的，用户很可能充值 以太坊的ERC20 上，如果您误选了ERC20，并且已经在metamask钱包中导入了BSC主网账号的情况下，可以通过下面的步骤把token找回来：

步骤​

  • 网络类型切换为太坊主网

  • 点击Add Token

  • 选择添加您提现时的币种，如USDT

  • 添加后，提现的币就在您这个账号里

  • 重新充值

你通过 metamask重新充值回币安主站，注意：充值方式是 ERC20
